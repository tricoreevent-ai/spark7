## 🎓 Basic Concepts

### What is a commit?
A "save point" for your code with:
- A description of what changed
- Who made the change
- When it was made
- All the actual code changes

### What is a push?
Uploading your commits to GitHub so:
- Your code is backed up
- Your team can see the changes
- It's available everywhere

### What is a branch?
A separate version of your code where you can:
- Develop new features safely
- Fix bugs without affecting main code
- Experiment freely
